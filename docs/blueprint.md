# **App Name**: DesignFlow Portfolio

## Core Features:

- Portfolio Categorization: Organize portfolio items into categories like branding, social media designs, and posters.
- Project Pages: Dedicated pages for each project displaying multiple designs (images).
- Full-Screen Image View: Enable users to view each design (image) in a project in full-screen mode.
- Contact Form: A contact form to allow potential clients to reach out.

## Style Guidelines:

- Primary color: Orange (#FF7010), conveying energy and creativity.
- Background color: White (#FFFCFC) to provide a clean and neutral backdrop, ensuring the designs stand out.
- Secondary color: Black (#101010) for text and supporting elements to add contrast and sophistication.
- Font pairing: 'Bricolage Grotesque' or 'Lufga' (sans-serif) for headings, providing a modern and artistic feel; a readable serif font for body text to ensure readability in longer descriptions.
- Minimal, line-based icons to represent services and categories.
- Clean, grid-based layout with ample white space to showcase designs effectively.
- Subtle transition animations for page navigations and image loading to enhance user experience.
